---------------------------------------------------------------------------------------------------------
Purpose:
---------------------------------------------------------------------------------------------------------
    Demonstrate how to control SPI flash via SPI interface.
		
---------------------------------------------------------------------------------------------------------
Operation:
---------------------------------------------------------------------------------------------------------
    (1) The project can be directly demonstrated on the "EVB-I91500ADI V1.0" board.
    (2) The SPI interface used in this project is "PA12=MOSI0, PA13=CLK, PA14=MISO0, PD8= SS0".
    (3) Compiled to execute.
    (4) Program test procedure -      
        1.Press enter(semihost or UART) to start SPI flash detection.
        2.if successful will be shown SPI Flash erase/program/read sample end...
